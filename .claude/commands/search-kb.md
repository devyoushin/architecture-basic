---

`$ARGUMENTS` 키워드로 지식 베이스를 검색합니다.

## 검색 범위
- 프로젝트 루트의 모든 `*.md` 파일 (01~20번 문서)

## 출력 형식

```
## 검색 결과: "{키워드}"

### 관련 문서
| 파일 | 관련 섹션 | 연관도 |
|------|---------|-------|
| {파일명} | {섹션명} | High/Medium/Low |

### 관련 아키텍처 레이어
{키워드가 어떤 아키텍처 레이어에 속하는지 (거버넌스/네트워크/보안/운영/플랫폼)}
```
